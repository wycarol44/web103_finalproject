# User Stories

## User Roles

- Community Member - Submits cases, evidence, and arguments; votes where eligible; gains XP.
- Juror - A Community Member randomly selected to serve on a jury during the jury phase.
- Judge - A Community Member assigned to publish the final legal status of an object after the verdict.

## User Stories

Per case
|  User role | Desire | Reason |
| -------- | -------- | -------- |
| Community Member | submit an object with a short accusation (optional image) | the object can enter the provisional phase for community consideration |
| Community Member | browse all open cases grouped by phase | decide where to participate |
| Community Member | see the current phase and remaining time for every active case | I know when I can still contribute |
| Community Member | vote on provisional cases | promising cases advance to trial |
| Community Member | submit evidence (optional image) | support or undermine an accusation |
| Community Member | vote on the quality of submitted evidence | the most compelling evidence is highlighted |
| Community Member | see the highest-rated evidence and arguments first | I can quickly understand the strongest positions in a case |
| Community Member | submit an argument citing evidence and/or previous cases | persuade the court |
| Community Member | view arguments together with their cited evidence and precedents | evaluate its reasoning |
| Juror | be randomly assigned eligible cases during the jury phase | verdicts are decided by a representative jury |
| Juror | cast a verdict of "Guilty" or "Not Guilty" | the trial reaches a conclusion |
| Judge | declare the object's final legal status after the jury reaches a verdict | the case is officially closed |
| Community Member | see which later cases reference a completed case | I can follow evolving bird jurisprudence |
| Community Member | view completed cases with every phase preserved | I can read the entire history of the trial |

User profiles
|  User role | Desire | Reason |
| -------- | -------- | -------- |
| Community Member | see all current and past cases I have contributed to | I can track my participation |
| Community Member | earn XP for submitting content and voting | my contributions are recognized |
| Community Member | see how much XP I earned from each action and vote | I understand which contributions were valuable |
| Community Member | the system to enforce daily submission and voting limits | participation remains fair and varied |
| Community Member | create and edit a profile with a name, profile image, and optional bio | other users can learn about my courtroom persona and my contributions are visually identifiable throughout the site |
| Community Member | choose which achievements are highlighted on my profile | I can showcase the accomplishments I value most |
| Community Member | view all of my earned achievements | I can track my long-term progress |
| Community Member | view another user's profile + summary statistics | I can see their bio, highlighted achievements, and overall participation |
| Community Member | achievements to include descriptions and unlock criteria | I understand how each achievement was earned or can be earned in the future |
| Community Member | see a timeline of another user's recent contributions | I can easily find cases, evidence, and arguments they have participated in |
| Community Member | earn achievements for specific milestones or unusual accomplishments | participation rewards more than simply accumulating XP |
